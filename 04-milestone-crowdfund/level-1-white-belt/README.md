# Milestone Crowdfund - Level 1 White Belt

## Goal

Build the first working Stellar Testnet dApp for milestone-based crowdfunding.

## Requirements

- Freighter wallet setup
- Stellar Testnet connection
- Wallet connect and disconnect
- XLM balance fetch and display
- XLM transaction send flow
- Success/failure feedback with transaction hash
- Error handling for wallet, balance, and transaction states
